# Ejercicio 02 del SDP 25/26

## Descripción

Esta interfaz recrea una pantalla de una aplicación del tiempo con diseño adaptado a iOS 26 usando Swift 6 y SwiftUI.  
Un sol con nubes en la parte superior. Debajo, una imagen con plantas y la temperatura en letra con sombra. Debajo, tres boxes con una distribución vertical con distintos elementos sobre la previsión por días. Debajo una pila de pequeñas cajas con previsión y temperatura.

## Objetivos del ejercicio

- Practicar el uso de imágenes como cabecera con `resizable()` y `scaledToFill()`.
- Colocar contenido sobre zonas de isla dinámica usando `safeAreaInset` y capas (`ZStack`).
- Implementar un scroll horizontal para ver el resumen de los días posteriores.
- Crear pickers y progressview.
- Seguir patrones de diseño de iOS 26 (Liquid Glass, jerarquía visual clara, márgenes amplios).
